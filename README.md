# counting-it
Цикл while в методе main должен отработать ровно четыре раза, чтобы на экран вывелись 4 строки.
Изменения можно вносить только в метод amazingMethod.

псевдокод:

```
ПСЕВДОКОД:

ПРОГРАММА "Нужно просто посчитать"

НАЧАЛО_ПРОГРАММЫ
    ОБЪЯВИТЬ localTime = время_полуночи
    ОБЪЯВИТЬ next = вызвать_удивительный_метод(localTime)
    
    ПОКА next после localTime ВЫПОЛНЯТЬ:
        ВЫВЕСТИ next на экран
        next = вызвать_удивительный_метод(next)
        ЗАДЕРЖКА 500 миллисекунд
    КОНЕЦ_ПОКА
КОНЕЦ_ПРОГРАММЫ

ФУНКЦИЯ вызвать_удивительный_метод(базовое_время)
    ВОЗВРАТ базовое_время + 288 минут
КОНЕЦ_ФУНКЦИИ
```

Пояснение работы программы:

1. Начинаем с полуночи (00:00)
2. Каждый раз добавляем 288 минут (4 часа 48 минут)
3. Выводим каждое новое время с задержкой 0.5 секунды
4. Цикл продолжается, пока полученное время больше предыдущего

Пример вывода:

```
04:48
09:36
14:24
19:12
00:00 (на этом цикл остановится, так как 00:00 не после предыдущего времени)
```

Программа делает 5 итераций, так как 288 × 5 = 1440 минут = 24 часа, то есть возвращается к полуночи.
